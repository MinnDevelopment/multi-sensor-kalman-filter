
Installation: python3 -m pip install -r requirements.txt

Das Program fragt interaktiv nach bestimmten Einstellungen für die Simulation.

Ausführen:

- *NIX: "./main.py" oder "python3 main.py"
- Windows: "python3 main.py" oder "py main.py"
- Alternativ mit PyCharm oder anderen IDEs

Optionen:

Rader:
 Verwendet Range/Azimuth sensoren mit an verschiedenen Positionen.
 Das "preset" sind 4 sensoren in nah am Zentrum der Flugbahn.
 Die Sensoren sind durch ein X makiert.

 Falls nicht das preset verwendet wird können unbegrenzt viele Sensoren mittels x/y positionen übergeben werden.
 Sinnvoll ist es dabei zwischen -10000 und +10000 zu bleiben.

Grid:
 Verwendet einen Kartesischen sensor.

Retrodiction kann nach der Auswahl des Sensors abgestellt werden.
